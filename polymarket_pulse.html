Help

Polymarket Pulse is a browser-only research dashboard. It never places trades, never connects wallets, and never asks for Polymarket credentials.

API limitations
Gamma, CLOB, Data API, RSS, and WebSocket endpoints may change shape, rate-limit, or fail CORS checks. Each provider failure is logged and isolated so the rest of the UI keeps working.
Browser key risk
OpenAI calls run from your browser. Remembered API keys are stored in localStorage, which is convenient but not secure storage.
Research-only labels
LLM output is constrained to educational classifications such as watch, possible mispricing, news-sensitive, too wide, avoid, and stale.
RSS proxy behavior
Many news sites block browser RSS fetches. Configure a proxy template containing {url} to fetch those feeds.
