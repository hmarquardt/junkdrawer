Debug / About

Local IndexedDB stores coins, price ticks, snapshots, RSS news, LLM history, and logs. localStorage stores settings and the OpenAI key only if chosen. CORS can block RSS feeds; proxy settings are configurable. Exchange symbols may not exist for every coin.

Real-time WebSocket is optional. Coinbase and Kraken are usually better US-friendly public market feeds, though not every coin/pair is listed everywhere. Binance.com streams can fail because of region, network, VPN, firewall, or browser policy. CoinGecko polling remains the reliable default path for this static page.

This is a personal research dashboard, not financial advice. It never places trades and never connects to wallets or trading exchanges.
