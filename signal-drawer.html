About Signal Drawer

Signal Drawer is a personal RSS/topic tracker that runs entirely in your browser. It fetches configured feeds, filters by enabled key terms, caches articles, and lets you save or hide items locally.

Developer notes

This is local-only. Sources, terms, cache, read/saved/hidden state, and settings are stored in localStorage.

Feeds may fail because direct browser requests to RSS feeds often hit CORS restrictions. The default proxy is https://api.allorigins.win/raw?url=${encodeURIComponent(feedUrl)}, but it is editable in Admin.

If a source test says the response is HTML instead of RSS/Atom, the URL is probably a normal web page, a 404 page, or a site-rendered error page. Open the URL directly and look for XML beginning with <rss, <feed, or feed items.

A private Cloudflare Worker, Netlify Function, or AWS Lambda proxy is better long-term because public proxies can be slow, unavailable, or rate-limited.

localStorage is not secure storage. Do not store secrets, private tokens, or sensitive source URLs here.

Term syntax

Use plain terms like OpenAI, quoted phrases like "artificial intelligence", and excluded terms beginning with minus like -sports. Matching is case-insensitive.
